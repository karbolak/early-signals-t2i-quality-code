# Artifacts

This directory contains the data artifacts that document the experiment.

Logs can be found in trajectory_runs while the final scored dataset can be found in final_dataset.

The uncompressed CSV is the primary artifact because it is easy to inspect directly. The compressed copy is included only to make downloading easier and to allow checksum verification.

Raw chunk folders, generated images, and intermediate analysis directories are intentionally not included here because the final scored dataset already contains the relevant merged information needed to inspect and rerun the thesis analyses.
